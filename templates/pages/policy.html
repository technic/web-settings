What is web-settings and how it works?

Here we give a brief overview of the IPtvDream 4X web-settings structure and corresponding data flow.

Web settings is a web interface for IPtvDream 4X plugin ("Application"), which is an application for set-top-box enigma2 devices ("Device"). It allows to configure the Application online with help of a web interface. All the configuration data you enter on this website is sent back directly to the Application. The web-settings server itself hold your data only for a short time, required for the synchronization with the Device.

The settings of the Application can be categorized in two types:

  • Local settings, such as Application usability settings and player configuration. This data is stored on the Device locally, and is normally not further sent to the internet.
  • Provider settings, such as authorization data, streaming server configuration etc. This data is intended to be sent to the provider. We have no control of how this data is processed by the specific provider, so you have to consult with the privacy policy of the corresponding provider. The name of the provider and their company should be vivid from the Application user interface, and it is also shown on the web-settings input page. In case you have any questions, do not hesitate to contact us.
Technically the aforementioned settings usage is done by the Application, so you have to consult Application privacy policy for more details. The Website does not share data with any providers directly.

What data do we collect and how we use it?

The Website collects your data so that we can:

  • Monitor the usage of our services.
  • Detect, prevent and address technical issues.
  • Provide you with the web-settings interface functionality, by forwarding data to the Application. The Application, in turn, shares the data with other third party services (providers).

More detailed explanation of different types of data collection and processing is given below.

Usage Data

When you open the Website, our servers automatically record information that your browser sends. This data may include information such as your device's IP address, browser type and version, operating system type and version, language preferences or the webpage you were visiting before you came to the Website and Services, pages of the Website that you visit, access times and dates, and other statistics.

Information collected automatically is used only to identify potential cases of abuse and establish statistical information regarding the usage and traffic of the Website and Services. This statistical information is not otherwise aggregated in such a way that would identify any particular user of the system.

We retain Usage Data for internal analysis purposes. The usage data is stored in the standard web server log files and can be kept for up to 6 months. Once this time period has expired, we will delete your data by usual log rotation procedure.

Personal information

When using the web-settings Website you provide us with the settings data. Settings data can include logins and password for 3rd party services and specific device settings such as player configuration.

This data is forwarded to the Application, which can use it to access 3rd party services. Your consent to this Privacy Policy followed by your submission of such information represents your agreement to that transfer. This Privacy Policy doesn't cover how the target Device (or Application which uses this web settings interface) processes provided data.

The settings data is stored in the server cache and is sent to the target Device. As soon as the target Device receives the settings data it usually sends the command to the server to erase the data cached on the server. Otherwise the cache is erased from the server within 24 hours.
